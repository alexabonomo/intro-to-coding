Basic Javascript for Interactivity and Intro to p5.js

What is javascript?

Javascript is a lightweight programming language ("scripting language") that allows for interativity, using data, performing calculations, and changing HTML elements on the page.

The DOM is a programming interface for web documents.
It represents the page so that programs can change the document structure, style, and content. 
The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

Intro to p5.js

p5.js is a JavaScript library for creative coding, with a focus on making coding accessible and inclusive for artists, designers, educators, beginners, and anyone else! p5.js is free and open-source because we believe software, and the tools to learn it, should be accessible to everyone.

Lets take a look at the basics: https://editor.p5js.org/
