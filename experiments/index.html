Glossary

  • generative design
    • An iterative design process that generates outputs that meet specified constraints to varying degrees. The output can be items such as images, sounds, architectural models, animation, and industrial parts. It is used in design fields such as art, architecture, communication design, and product design.
  • library
    • A library is a collection of code that does a job (or bunch of related jobs). We use a few libraries in this module: p5.js and THREE.js.
  • 3D photogammetry
    • The process of taking many photographs of an object from a variety of angels and stitching them together to create a 3D model. We use the Luma AI application to do this for the 3D on the Web example.
