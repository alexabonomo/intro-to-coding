Hover States

Last part, we looked at hover states by adding a selector with a psuedoclass of :hover

for example:

HTML

<div class="hover-item"></div>

CSS

.hover-item{
        background-color: #3825ae;
    }
    .hover-item:hover{
        background-color: #ae2563;
    }

If you want a more smooth transition, you can add the transition property.

for example:

.hover-item{
    background-color: #3825ae;
    transition:2s
}
.hover-item:hover{
    background-color: ae2563;
    transition: 2s;
}

If you want to change the size, you can add the scale property.

for example:

.hover-item{
    background-color: #3825ae;
    transform: scale(1);
    transition: .5s;
}
.hover-item:hover{
    transform: scale(.75);
    transition: .5s;
}

If you want to switch between images on hover, you can use the background properties.

for example:

.hover-item{
    background-color: #ffffff;
    background-image: url("../images/
    pink-praying-mantis.png");
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat; 
    transition: .5s;
}
.hover-item:hover{
    background-image: url("../images/
    cat.png");
    transition: .5s;
}
