Interactions, Input & Output

Lets talk about interactivity and how you can make your site become more dynamic and lively!

One common way to incorporate interaction is to add a hover state. You can do this by adding a selector with a psuedoclass of :hover

for example:

HTML

<div class="hover-item"></div>

CSS

.hover-item{
    background-color: black;
}
.hover-item:hover{
    background-color: hotpink;
}

If you want a more smooth transition, you can add the transition property.

for example:

.hover-item{
    background-color: black;
    transition:2s
}
.hover-item:hover{
    background-color: hotpink;
    transition: 2s;
}

If you want to increase the size, you can add the scale property.

for example:

.hover-item{
    background-color: black;
    transform: scale(1);
    transition: .5s;
}
.hover-item:hover{
    transform: scale(1);
    transition: .5s;
}

If you want to switch between images on hover, you can use the background properties.

for example:

.hover-item{
    background-color: #ffffff;
    background-image: url("../images/pink-praying-mantis.png");
    background-size: 100%;
    background-position: center;
    background-repeat: no-repeat; 
    transition: .5s;
}
.hover-item:hover{
    background-image: url("../images/cat.png");
    transition: .5s;
}

If you want to switch between images on hover, you can use the background properties.

for example:

.hover-item{
    display:flex;
    align-items: center;
    justify-content: center;
    color:white;
    background-size: 100%;
    background-position: center;
    background-repeat: no-repeat;
    transition: .5s;
}
.hover-item:hover{
    background-image: url("../images/bearhug.png");
    transition: .5s;
}
.hover-item:hover p{
    display: none;
}
